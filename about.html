My name is Joey Mason and I'm a software engineer and entrepreneur based in Los Angeles. I'm currently serving as CTO of Repost Network, which I co-founded with Jeff Ponchick in 2015. I hold a degree in Computer Science from California State University, Long Beach and have previously worked as a software engineer at FreedomPop and DirecTV.

Lately, I've been working a lot with Java 8 and Spring 4, as well as MongoDB, AngularJS and Elastic Beanstalk. I'm most excited about AWS Lambda and the Serverless Application Framework. It's an exciting time to be a developer!
